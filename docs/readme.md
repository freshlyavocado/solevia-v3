# Panduan Struktur File Solevia V3 🚀

Selamat datang di dokumentasi struktur proyek Solevia V3! 
Proyek ini menggunakan arsitektur modern **Headless E-commerce**, yang artinya bagian "Otak/Server" (Backend) dan bagian "Wajah/Tampilan" (Frontend) dipisah secara total agar lebih cepat, aman, dan mudah dikembangkan.

Silakan pilih panduan di bawah ini untuk memahami setiap bagian rumah kita:

1. [Struktur Backend (Laravel 13)](./struktur_backend.md) - Pelajari bagaimana server, database, dan API bekerja di balik layar.
2. [Struktur Frontend (Vue 3)](./struktur_frontend.md) - Pelajari bagaimana antarmuka pengguna, komponen, dan pengaturan *state* (data) dibangun.
